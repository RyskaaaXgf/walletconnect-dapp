# WalletConnect Demo App

This project is a simple Web3 application that integrates WalletConnect  
to allow users to connect their crypto wallets seamlessly.

## Features
- Connect wallet using WalletConnect
- Read wallet address
- Sign messages
- Simple and clean UI

## Tech Stack
- WalletConnect SDK
- JavaScript
- HTML / CSS

## Purpose
This project was built for learning and experimenting with WalletConnect  
as part of the WalletConnect Builders program.
